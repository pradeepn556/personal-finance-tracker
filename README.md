## 🎯 What This App Does

| Feature | Description |
|---|---|
| **Dashboard** | Net worth trend, cash flow charts, financial health metrics |
| **Income** | Track multiple income sources (salary, freelance, etc.) with charts |
| **Investments** | Portfolio P&L tracking, holdings heatmap, asset allocation |
| **Expenses** | Budget tracking per category, spending heatmap, trends |
| **Settings** | Export/import data as JSON or CSV, display preferences |

## 🗂 Project Structure

```
src/
├── components/
│   ├── Dashboard/       # Net worth, cash flow, summary cards
│   ├── Income/          # Income entry, charts, table
│   ├── Investments/     # Portfolio, P&L, heatmap
│   ├── Expenses/        # Budget tracking, spending analysis
│   ├── Settings/        # Export/import, preferences
│   └── shared/          # Reusable UI components
├── utils/
│   ├── storage.js       # localStorage read/write helpers
│   ├── calculations.js  # Net worth, P&L, budget formulas
│   └── formatters.js    # Currency (AUD), date formatting
└── App.jsx              # Main app with tab navigation
```

## 🚀 Run Locally

```bash
git clone https://github.com/pradeepn556/personal-finance-tracker.git
cd personal-finance-tracker
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
